    伊水河 2021/1/4 leetcode算法

    十-二进制数的最少数目 (opens new window)

    消耗：

    脑筋急转弯，真的是。。。

    拆解示例：

    32
    11
    11
    10
    82734
    11111
    11111
    10111
    10101
    10100
    10100
    10100
    10000

    ....

    所以其实就是找数字字符串中的最大数字。

    /**
 * @param {string} n
 * @return {number}
 */
var minPartitions = function(n) {
    var max=0
    for(var i=0;i<n.length;i++){
        var t=n[i]
        if(t==9) return 9
        if(max<t){
            max=t
        }
    }
    return max
};
