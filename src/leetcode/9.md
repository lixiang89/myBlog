};

![消耗](/images/leetcode/9-2.png)

类似第七题。以121举例，除以10得12.1,整数部分为12，小数部分为1，用取余得小数，Math.floor得整数。

变量n用来记录反转的小数，x1复制一份原始数字x，开始循环：

|循环次数	|小数部分|	整数部分|	n(乘10加小数部分)|
|-	|-	|-	|-|
|1	|1	|12	|1|
|2	|2	|1	|12|
|3	|1	|0	|121|

```javascript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    if(x<0) return false
    var n=0,x1=x
    while(x){
        n=n*10+x%10
        x=Math.floor(x/10)
    }
    return x1===n
};
```
